======================================================
  광휘고 교직원 검색 앱 - 설치 방법 안내
======================================================

▶ 이 앱은 "PWA(설치형 웹앱)" 방식입니다.
   앱스토어 없이 바로 설치할 수 있습니다.

------------------------------------------------------
[STEP 1] 앱 파일을 인터넷에 올리기 (1회만)
------------------------------------------------------
이 폴더(광휘고앱)에 있는 파일 4개를 웹에 올려야 합니다:
  - index.html
  - manifest.json
  - sw.js
  - icon-192.png
  - icon-512.png

▶ 무료 호스팅 추천: GitHub Pages
   1. https://github.com 무료 가입
   2. 새 저장소(repository) 생성
   3. 파일 4개 업로드
   4. Settings → Pages → Source: main 선택
   5. 주소 생성 예: https://아이디.github.io/저장소명/

▶ 대안: 학교 홈페이지 서버의 빈 폴더에 업로드

------------------------------------------------------
[STEP 2-A] Android 설치 방법
------------------------------------------------------
1. 스마트폰 Chrome 브라우저로 앱 주소 접속
2. 주소창 오른쪽 ⋮ (점 세 개) 메뉴 탭
3. "홈 화면에 추가" 또는 "앱 설치" 선택
4. "설치" 버튼 탭
→ 홈 화면에 앱 아이콘이 생성됩니다!

------------------------------------------------------
[STEP 2-B] iOS(아이폰) 설치 방법
------------------------------------------------------
1. 스마트폰 Safari 브라우저로 앱 주소 접속
2. 하단 공유 버튼(□↑) 탭
3. "홈 화면에 추가" 선택
4. 이름 확인 후 "추가" 탭
→ 홈 화면에 앱 아이콘이 생성됩니다!

------------------------------------------------------

------------------------------------------------------
